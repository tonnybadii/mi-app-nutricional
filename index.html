📋 Configuración de Google Sheets

⚠️ Importante: Para usar esta app necesitas configurar Google Sheets para almacenar los datos de forma segura.

Paso 1: Crear Google Sheet

1. Ve a sheets.google.com

2. Crea una nueva hoja llamada "NutriPro-DB"

3. Crea 3 pestañas: "Alimentos", "Pacientes", "Consultas"

Paso 2: Configurar API

1. Ve a console.developers.google.com

2. Crea un proyecto nuevo

3. Habilita "Google Sheets API"

4. Crea credenciales (API Key)
